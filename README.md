# 项目详情

为提升MYSQL数据库的访问瓶颈，除了在服务器端增加缓存服务器缓存常用的数据之外（例如redis），还可以增加连接池，来增加MySQL的访问速度，以提升MySQL的访问效率，在高并发情况下，大量的TCP三报文握手、MySQL Server连接认证、MySQL Server关闭连接回收资源和TCP四报文挥手所消耗的性能时间也是非常明显的，增加连接池就是为了减少这一部分的损耗。
